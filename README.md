# Survivor — 像素幸存者

一个基于 Unity 的 **2D 像素动漫风 Vampire Survivors-like Rogue-lite 游戏**，拥有丰富的装备收集、多角色皮肤、技能进化树和社群 Boss 玩法。

---

## 项目规模

| 类别 | 数量 |
|------|------|
| 技能系统 | 11 个技能 + 3 个 UR 进化 |
| 角色/皮肤 | 4 个（琪诺露、南筱风、夏无、无罪） |
| Boss 类型 | 10 种（5 基础 + 4 社群 WorldBoss + 1 最终龙王） |
| 难度配置 | 14 个（N1 ~ N13 + 无尽模式） |
| 普通小怪 | 4 种（蘑菇人、蝙蝠、狼人、史莱姆） |
| 装备总数 | ~76 件（成就/通关/抽卡/好感度/继承） |
| 奇遇事件 | 10 个 |
| 社群 | 4 个（蘑菇/蝙蝠/狼人/史莱姆好感度） |

## 游戏特色

### 核心玩法
- **自动战斗 + 走位** — 技能自动向最近敌人释放，玩家只需专注于走位和生存
- **升级三选一** — 每次升级弹出三张卡牌：学习新技能 / 强化已有技能 / 提升人物属性；首轮保底全为学习卡
- **鼠标点击移动** — 支持 WASD/方向键 和鼠标左键点击地面移动，N1 开局自动弹出新手教程
- **倍速系统** — 1x/2x/3x 三档切换（3x 需成就装备解锁）
- **冲刺** — 空格键冲刺，可升级无敌/穿怪

### 养成系统
- **抽卡系统** — R / SR / SSR / UR 四档抽卡，软保底机制，SSR 直接影响局内战斗（解锁技能、进化路线、被动属性）
- **装备系统** — 四种装备类型（成就/通关/抽卡/好感度），持久化解锁，跨局生效；存档界面可查看获得条件
- **好感度社群** — 四个社群（蘑菇/蝙蝠/狼人/史莱姆），击败世界 Boss 积累好感度，分 10/50/100 三档解锁装备和被动加成
- **源木与奇遇** — 局内货币源木，消耗 200 触发随机奇遇二选一/三选一，效果包括临时增益、永久加成、技能强化

### 技能进化
- **风之形** — 飓风 + 风箭 → 永久环绕风刃
- **地狱火** — 火球术 + 风箭 → 从天而降的三叉戟
- **亡者领域** — 孢子领域 → 击杀敌人概率复活为永久友军
- **命途:寄生** — 狼人社群好感度解锁，发射寄生触手吸取生命

### 挑战内容
- **14 个难度** — N1 教学局到 N13 终极龙王 + 无尽挂机模式
- **10 种 Boss** — 蘑菇人/蝙蝠/狼人/史莱姆/龙王及对应社群世界Boss
- **门挑战** — N5+ 解锁的 13 层递增挑战，每层通关所有技能升级上限永久 +1，全通额外经验效率 +10
- **世界 Boss** — N7+ 开局生成，击败解锁对应社群
- **营地攻占** — 战场中出现可占领的中立营地
- **对局总结** — 结束后展示技能伤害占比、击败 Boss 列表、解锁装备等详细数据

### 其他
- **四位角色皮肤** — UR 角色通过抽卡解锁，各有独特被动与开局自带技能
- **读档复活** — R 级抽卡装备死亡时消耗 1 张满血复活
- **自动模式** — 成就装备 8「不可视之手」解锁，自动选取升级
- **装备积分** — 每次抽卡获积分，可兑换通关装备；无尽模式每分钟额外 +5
- **七彩蘑菇/孢子异变** — 蘑菇敌人 50% 概率随机变色

## 技术栈

- **引擎**：Unity 2022.3+ (C#)
- **渲染**：2D Sprite + SpriteRenderer
- **摄像机**：透视投影，45° 俯视视角
- **物理**：Unity Physics（3D Rigidbody + BoxCollider/SphereCollider）
- **UI**：Unity uGUI + TextMeshPro (heiti SDF 中文字体)
- **素材**：像素动漫风精灵图（chibi Q 版比例，~2.5 头身）
- **持久化**：PlayerPrefs（装备/好感度/难度解锁/教程标记）
- **摄像机**：Cinemachine CinemachineBrain

## 构建与运行

1. 使用 Unity Hub 打开项目（推荐 Unity 2022.3 LTS+）
2. `File → Build Settings` 选择目标平台
3. Editor 中可直接 Play 测试
4. 首次运行建议从 N1 难度开始，自动弹出新手教程
