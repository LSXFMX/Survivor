# Survivor — 像素幸存者

一个基于 Unity 的 **2D 像素动漫风吸血鬼幸存者类 Rogue-lite 游戏**，拥有丰富的抽卡养成系统、多角色皮肤、进化技能树和社群 Boss 玩法。

---

## 项目规模一览

| 类别 | 数量 |
|------|------|
| 技能系统 | 10 个技能预制体 + 3 个 UR 进化 |
| 角色/皮肤 | 4 个（琪露诺、南筱风、夏无、无罪） |
| Boss 类型 | 10 种（5 基础 + 4 社群 WorldBoss + 1 最终龙王） |
| 难度配置 | 14 个（N1 ~ N13 + 无尽模式） |
| 普通小怪 | 4 种（蘑菇人、蝙蝠、狼人、史莱姆） |
| 装备总数 | ~76 件（成就/通关/抽卡/好感度/继承） |
| 奇遇事件 | 10 个 |
| 社群 | 4 个（蘑菇/蝙蝠/狼人/史莱姆好感度） |

---

## 游戏特色

- **抽卡养成系统** — R / SR / SSR / UR 四档抽卡，软保底机制，抽卡装备可直接影响局内战斗（解锁新技能、进化路线、被动属性）
- **UR 进化技能** — 三大 UR 进化路线：风之形（飓风→风之形）、地狱火（火球+风箭→地狱火）、亡者领域（孢子领域→亡者领域）
- **四位角色** — 琪露诺（基础）、南筱风（风系UR）、夏无（火系UR/血族）、无罪（亡者领域UR），各具独特被动与开局加成
- **14 个难度** — N1 新手村到 N13 终极龙王，外加无尽挂机模式（正计时，每 5 分钟刷社群 Boss，每分钟送装备积分）
- **10 种 Boss** — 蘑菇人 / 蝙蝠 / 狼人 / 史莱姆 / 龙王 及对应的社群世界 Boss
- **亡者领域复活** — 无罪专属：孢子领域击杀的敌人有概率复活为永久友军，社群 Boss 也可复活为永久盟友
- **门挑战系统** — 13 层逐层挑战（每层怪物递增），每层通关随机属性奖励，全通额外经验效率 +10
- **10 个奇遇事件** — 消耗源木触发二选一/三选一抉择，包括人格解离（分身）、源木收集者（money is power）、愚弄（重置门挑战翻倍难度）等
- **自动模式** — 成就装备 8「不可视之手」解锁，自动选取升级（0.5s 高亮延迟），优先级：学习技能 > 技能升级 > 人物升级
- **装备积分系统** — 每次抽卡获积分，可兑换通关装备；无尽模式每分钟额外 +5 积分
- **营地攻占** — 战场中出现可占领的中立营地
- **冲刺 + 无敌帧** — 蘑菇滑板解锁冲刺，虚空斗篷 SSR 提供冲刺无敌穿透
- **读档复活** — R 级抽卡装备"读档币"，死亡时消耗 1 张满血复活
- **好感度社群** — 击败不同种族怪物积累好感度，解锁专属宠物与被动加成
- **三倍速** — 成就装备 4「沙漏」解锁
- **七彩蘑菇/孢子异变** — 蘑菇敌人 50% 概率随机变色

## 技术栈

- **引擎**：Unity (C#)
- **渲染**：2D Sprite + SpriteRenderer
- **物理**：Unity Physics（BoxCollider / Rigidbody kinematic）
- **UI**：Unity uGUI + TextMeshPro
- **素材**：像素动漫风精灵图（chibi Q 版比例，~2.5 头身）

## 构建与运行

1. 使用 Unity Hub 打开项目（推荐 Unity 2022.x+）
2. 确保安装 Android Build Support（如需移动端打包）
3. `File → Build Settings → Android → Switch Platform`
4. Editor 中可直接 Play 测试
